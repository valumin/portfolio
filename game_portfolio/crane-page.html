Crane Physics

Get on! fully working crane physics. Smoothly pickup and drop attachable objects!

description

If this game was to be published, I'd probably name it The Game of Cranes! because why not. I worked on this game for a couple weeks then threw it in the "MightContinueLater" folder. My main purpose was to model a crane, and make it work, that's all.

So far, the crane is working perfectly, I wouldn't want anything changed. And there is a character controller which moves Morty around. He's able to run, jump, fall and move objects around.
