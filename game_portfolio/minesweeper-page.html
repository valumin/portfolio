MineSweeper 3d

The same Minesweeper you know, but more fancy.

description

I'm a big fan of the original Minesweeper, I still play it on my phone, or in the labs during my university lectures when it gets boring.

This game was made out of curiousity "What would Minesweeper look like in 3D?".

I'll keep things simple. I wrote a piece of code that generates a grid ground of cubes, it's totaly customizable (grid size, tile size). Then i made a character controller to move the character around. Whenever the character steps into a grid box, it gets highlighted, then the player can either place a flag or dig up the grid box. everything that happens in the original Minesweeper is applied here.
